stn-core

The Foundation

Shared APIs, events, and utilities for all STN mods.

REQUIRED BY ALL

API Interfaces

ISurvivalNightProvider

Query survival night state from any mod.

isSurvivalNightActive()     // Is event running?
getDaysUntilSurvivalNight() // Days until next
getSurvivalNightInterval()  // Default interval
forceStart() / forceStop()  // Admin controls
getMobsSpawned()            // Current horde count
getTargetHordeSize()        // Target for tonight

IGamestageProvider

Access gamestage progression data.

getWorldGamestage()              // Average of all players
getPlayerGamestage(player)       // Specific player
getPlayerGamestage(uuid)         // By UUID
addGamestage(player, amount)     // Grant points
removeGamestage(player, amount)  // Penalty
getHordeSizeMultiplier()         // 1.0 to 2.5

ISoundEmitter

Register sounds for zombie sensory system.

emitSound(world, pos, volume, type)  // Alert zombies
SoundType: BLOCK_BREAK, DOOR, COMBAT, EXPLOSION, PLAYER_ACTION

IDurabilityProvider

Query and modify block durability.

getDurability(world, pos)           // Current HP
getMaxDurability(world, pos)        // Max HP
damage(world, pos, amount)          // Deal damage
repair(world, pos, amount)          // Heal block

Events

STNEvents.SURVIVAL_NIGHT_START  // (world, gameDay)
STNEvents.SURVIVAL_NIGHT_END    // (world, mobsSpawned, survived)
STNEvents.GAMESTAGE_CHANGED     // (player, oldStage, newStage)
STNEvents.SOUND_REGISTERED      // (world, pos, volume, type)
STNEvents.BLOCK_DAMAGED         // (world, pos, damage, remaining)
STNEvents.BLOCK_REPAIRED        // (world, pos, amount, newDurability)
STNEvents.BLOCK_DESTROYED       // (world, pos)
