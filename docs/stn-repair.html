Hammer Tools for Block Repair

Five tiers of hammers to repair damaged blocks. HUD shows durability when aiming.

Depends: stn-core Depends: stn-fortifications

Hammer Tiers

Wooden Hammer

Durability: 59 uses

Repair: 10 HP per use

Sticks + Planks. Basic but gets the job done.

Stone Hammer

Durability: 131 uses

Repair: 15 HP per use

Sticks + Cobblestone. More durable, slightly better repair.

Iron Hammer

Durability: 250 uses

Repair: 25 HP per use

Iron Nuggets + Iron Ingot. The workhorse hammer.

Diamond Hammer

Durability: 1561 uses

Repair: 40 HP per use

Iron Ingots + Diamond. Long-lasting, efficient repair.

Netherite Hammer

Durability: 2031 uses

Repair: 60 HP per use

Diamond Hammer + Netherite (Smithing Table). Best in class.
