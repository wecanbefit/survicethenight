Survival Night Events & Gamestage Progression

The core event system. Every 6 days, hordes spawn in waves until dawn. Gamestage progression scales difficulty based on your survival.

Depends: stn-core

Configuration

Timing Settings

Setting Default Description
SURVIVAL_NIGHT_INTERVAL 6 Days between events (~2 hours real time)
SURVIVAL_NIGHT_START_HOUR 22 In-game hour event starts (10 PM)
SURVIVAL_NIGHT_END_HOUR 6 In-game hour event ends (6 AM)

Horde Settings

Setting Default Description
BASE_HORDE_SIZE 20 Starting horde size at day 1
HORDE_SIZE_MULTIPLIER_PER_DAY 0.1 +10% more zombies per day survived
MAX_HORDE_SIZE 200 Maximum zombies in a single night
SPAWN_RADIUS_MIN 24 Minimum spawn distance from player
SPAWN_RADIUS_MAX 48 Maximum spawn distance from player
SPAWN_WAVE_INTERVAL 600 Ticks between waves (30 seconds)

Difficulty Scaling

Setting Default Description
SCALE_WITH_PLAYER_COUNT true Scale horde for multiplayer
PLAYER_COUNT_MULTIPLIER 0.5 +50% horde per additional player
SURVIVAL_NIGHT_SPEED_MULTIPLIER 2.0 Zombie speed during event (2x normal)
SURVIVAL_NIGHT_DAMAGE_MULTIPLIER 1.5 Zombie damage during event (1.5x)

Gamestage Formula

GAMESTAGE_SURVIVAL_NIGHT_BONUS 5 +5 gamestage per survival night
GAMESTAGE_DEATH_PENALTY 2 -2 gamestage per death

API Usage

Checking Survival Night State

// Check if survival night is active
boolean active = STNCore.isSurvivalNightActive();

// Get days until next survival night
int daysLeft = STNCore.getSurvivalNightProvider().getDaysUntilSurvivalNight();

// Get current world gamestage
int gamestage = STNCore.getGamestageProvider().getWorldGamestage();

// Get player-specific gamestage
int playerGS = STNCore.getGamestageProvider().getPlayerGamestage(player);

Listening to Events

// When survival night starts
STNEvents.SURVIVAL_NIGHT_START.register((world, gameDay) -> {
    // Trigger your effects here
});

// When survival night ends
STNEvents.SURVIVAL_NIGHT_END.register((world, mobsSpawned, survived) -> {
    // Award bonuses, play sounds, etc.
});

// When gamestage changes
STNEvents.GAMESTAGE_CHANGED.register((player, oldStage, newStage) -> {
    // Notify player of milestone
});
